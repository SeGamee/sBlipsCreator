
# sBlipsCreator

Un script permettant de créer et gérer vos blips rapidement et simplement, aperçu de votre blips lors de la création en temps réel pour faciliter la création.


## Requierement

 - [ESX Legacy]
 - [oxmysql] ou [mysql-async]
